# Hostmate AI — Landing Page

Landing page moderne pour Hostmate AI, construite avec Vite + React + TypeScript + Tailwind CSS.

## Scripts

- `npm run dev` — démarrer le serveur de dev
- `npm run build` — build de production
- `npm run preview` — prévisualiser le build

## Structure

- `src/App.tsx` — page unique (hero, features, stats, testimonials, pricing, FAQ, CTA, footer)
- `src/index.css` — styles globaux + Tailwind

## Personnalisation

Modifiez les textes et sections directement dans `src/App.tsx`.
